# 🌾 SS Flour Mill Website

A modern and responsive **Atta (Flour) Delivery Website** built using HTML, CSS, and JavaScript.  
This project allows users to easily calculate price and order fresh atta directly via WhatsApp.

---

## 🚀 Live Website
https://solanki-tarun.github.io/ss-flour-mill/

---

## ✨ Features

- 🛒 Easy product selection (5kg, 10kg, etc.)
- 💰 Auto price calculation
- 📲 WhatsApp instant order system
- 📍 Google Maps location integration
- 📞 One-click call button
- 📱 Fully responsive (mobile friendly)
- ⚡ Fast and lightweight design

---

## 🧑‍💻 Tech Stack

- HTML5  
- CSS3  
- JavaScript
